# Savitri Sawmill

Welcome to the **Savitri Sawmill** website repository. This project is a modern, responsive web application built to showcase the sawmill's products, services, and gallery, providing an elegant and professional online presence.

## 🚀 Tech Stack

This project is built with a cutting-edge frontend stack to ensure optimal performance, SEO, and developer experience:

- **Framework:** [TanStack Start](https://tanstack.com/start) - A full-stack React framework.
- **UI Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) - Utility-first CSS framework for rapid UI development.
- **Components:** [shadcn/ui](https://ui.shadcn.com/) & Radix UI - Accessible and customizable UI components.
- **Icons:** [Lucide React](https://lucide.dev/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Forms & Validation:** React Hook Form + Zod

## 🗺️ Key Pages

The application features several key routes to serve customers:

- `/` - **Home**: The landing page introducing Savitri Sawmill.
- `/about` - **About Us**: Information about the company and its history.
- `/services` - **Services**: Details on the services offered.
- `/products` - **Products**: A catalog of available timber and wood products.
- `/gallery` - **Gallery**: Images showcasing past work, facilities, and products.
- `/contact` - **Contact**: Contact information and inquiry form.
- `/privacy` - **Privacy Policy**: Standard privacy policy page.

## 🛠️ Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine. We recommend using [Bun](https://bun.sh/) as the package manager.

### Installation

1. Clone the repository and navigate into the project directory.
2. Install the dependencies:

```bash
bun install
# or
npm install
```

### Development

To start the development server with hot-module replacement (HMR):

```bash
bun run dev
```

Open your browser and navigate to `http://localhost:5173`.

### Building for Production

To create an optimized production build:

```bash
bun run build
```

You can preview the production build locally with:

```bash
bun run preview
```

## 🧑‍💻 Scripts & Code Quality

This project enforces code quality and formatting through ESLint and Prettier.

- Run the linter: `bun run lint`
- Format the code: `bun run format`
